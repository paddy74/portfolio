## Job Description

As an **Undergraduate Research Assistant** with Old Dominion University's Computer Vision Lab, I contributed to NIH-funded biomedical imaging research and Army-funded security analytics by developing deep-learning models, computer-vision pipelines, and LiDAR-based behavior-recognition systems.

- Created Python- and C++-based extensions for the 3D Slicer biomedical-imaging software, enabling users to quickly and quantitatively measure and visualize classification results using custom or pre-trained inference models.

- Developed deep-learning computer-vision models for identifying abnormal (cancerous) brain tissue in MRI scans, applying pre-processing pipelines, multi-modal feature extraction, and transfer learning using TensorFlow and OpenCV.

- Built a LiDAR-based human gait and behavior-recognition pipeline for Army research, implementing feature fusion, multi-person tracking, and action-classification experiments.

- Established the lab's Git/GitHub usage and provided training guides and best practices, replacing ad-hoc code sharing practices with modern version-control standards.
